## 4. 주요기능 소개

##### 문성용
- 판매자 페이지에서 로그인 세션이 사라지면 로그인 페이지로 이동
- 고객센터를 웹소캣을 이용한 챗봇 연결로 실시간 문답 가능
- 네이버 OCR을 통한 사업자 등록증 입력
- 네이버 오브젝트 스토리지를 통해 파일 통합관리
- 가게 등록, 삭제, 수정
- 주문 확인, 수락, 거절, 배송 처리
- 가게 주문 완료 목록 확인
  
##### 배기진
- 관리자 회원가입, 수정, 탈퇴
- 메뉴의 카테고리와 옵션 등록 삭제
  
##### 윤찬빈
- 사용자 페이지 가게 검색, 메뉴 검색
- 메뉴 장바구니 등록, 삭제
- 장바구니 구매
  
##### 진가영
- 사용자 페이지 회원가입, 수정, 탈퇴
- 주문목록 확인
